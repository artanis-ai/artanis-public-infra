# artanis-infra

AWS CDK code for self-hosted Artanis. This repo is a manual mirror of the `infra/` directory in the private Artanis source repo, refreshed at each release tag.

Do not open PRs here — they will be overwritten by the next release. File issues against the source repo instead.

Customers: see the `self-hosting-customer.md` runbook Artanis provided for deployment instructions.
